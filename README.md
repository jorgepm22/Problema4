Problema 4:

Para poder levantar este ambiente es necesario las siguientes dependencias:
Node.js preferiblemente la version 10.16 y npm preferiblemente la version 6.9 ve a la pagina web (https://nodejs.org/en/) y descarga el instalador. Git (Si quieres clonar el repositorio) y finalmente Ionic CLI.

Una vez descargadas e instaladas todas las depedencias descarga o clona el repositorio. En la carpeta raiz del proyecto ejecuta el npm i para instalar las dependecias del proyecto. Una vez intaladas las dependencias, ejecuta ng serve para iniciar el ambiente en tu ordenador.
